# Student Grade Report Generator – Python Mini Project 📊

This project takes student marks for 3 subjects, calculates the total, percentage, and assigns a grade based on standard grading criteria.

## ✅ Features:
- Takes student name and marks
- Calculates total and percentage
- Assigns grades (A+, A, B, C, F)
- Prints a clean formatted report card

## 🧪 Example:
Name: Muhammad Qamar
English: 85
Math: 92
Science: 78
Total Marks: 255/300
Percentage: 85.0%
Grade: A+
