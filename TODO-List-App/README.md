# To-Do List App – Python Mini Project ✅

This is a text-based To-Do List app where users can:
- Add new tasks
- Mark tasks as done
- Delete tasks
- View the full task list

## 🔧 Features:
- Menu-based system
- Uses Python lists and dictionaries
- Simple and powerful
